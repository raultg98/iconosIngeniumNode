# ICONOS INGENIUM CON NODE.JS

# DESCARGA 
<p>
    Una vez que tenemos descargado el proyecto tenemos que poner los 
    siguientes comando en la consola
</p>

<ol>
    <li>cd /carpetaPoyecto/</li>
    <li>npm install</li>
    <li>cd src</li>
    <li>node app.js</li>
</ol>
